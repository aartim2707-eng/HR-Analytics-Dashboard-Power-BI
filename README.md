# HR Analytics Dashboard – Power BI

## 📌 Domain
Human Resources (HR) – Workforce Analytics

## 📌 Project Overview
This project leverages **Power BI** to analyze HR data and provide actionable insights into:
- Employee Attrition trends  
- Recruitment Funnel efficiency  
- Skill Gap Analysis  

The dashboards help HR teams make **data-driven decisions** about hiring, retention, and workforce planning.

## 📌 Objectives
- Identify key reasons for employee attrition  
- Track recruitment performance (hiring rate, offer acceptance, drop reasons)  
- Highlight skill gaps across positions and departments  
- Compare expected vs. offered salaries  
- Provide insights for HR managers to improve workforce strategies  

## 📊 Dashboards (Inside Details)

### 1. Employee Attrition Dashboard
- **Attrition Count & Rate** → 387 employees, 14% attrition rate  
- **Attrition by Department** → Highest in Production, followed by IT/IS and Software Engineering  
- **Attrition by Age Band** → Most attrition in 20–30 and 31–40 age groups  
- **Attrition by Gender** → Female attrition higher (60%) compared to male (40%)  
- **Attrition by Training Outcome** → Higher attrition among employees with incomplete or failed training outcomes  
- **Yearly Attrition Trends** → Increase from 2022 (~150) to 2023 (~250)  

---

### 2. Recruitment Funnel Dashboard
- **Hiring Rate** → 58.73% (881 hired out of 2K candidates)  
- **Offer Acceptance %** → 100% acceptance among candidates offered  
- **Drop Reasons** → Salary mismatch, skill mismatch, better offer, no show, position closed  
- **Hires by Source** → Balanced distribution across LinkedIn, Naukri, Referral, and Company Website (~24–26% each)  
- **Average Time to Hire by Department** → IT & Operations (22 days), Analytics & HR (21 days)  
- **Recruiter Performance** → Recruiters R1–R4 contributed between 150–200 hires each  

---

### 3. Skill Gap Analysis Dashboard
- **Avg Skill Match % vs. Skill Gap %** → Avg skill match ~71%, avg skill gap ~29%  
- **Salary Expectations vs. Offered Salaries** → Candidates expect ~9.66 LPA, offered salaries slightly lower  
- **Skill Gap by Position** → Higher gaps in Data Analyst, QA Tester, and HR Executive roles  
- **Skill Gap by Location** → Delhi, Hyderabad, and Mumbai show varying gap levels  
- **Candidate Rejection Reasons** → Skill mismatch (29%), better offer (25%), salary mismatch (23%), no response (23%)  

## 📌 Key Insights
- Attrition rate is higher among younger employees (20–30 age band)  
- Recruitment funnel shows **58% hiring rate**, with salary mismatch as a major drop reason  
- Skill gap analysis highlights **Data Analyst & QA Tester roles** with higher gaps  
- Salary expectations often exceed offered salaries, impacting acceptance rates
